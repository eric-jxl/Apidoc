# 巡检任务查询接口



**简要描述：**

- 查询巡检检查任务和异常任务数量

**请求URL：**

- `URL/api/get_task_numberr`

**请求方式：**

- GET

**参数：**

| 参数名   | 必选 | 类型   | 说明                              |
| :------- | :--- | :----- | --------------------------------- |
| token    | 是   | string | 密钥                              |
| uid      | 是   | Int    | 用户id                            |
| category | 是   | string | 可选项（checkTask，checkUnusual） |

**返回示例**

```json
{
	"code":200,
  "data":{
      "常规任务":1,
      "异常任务":1
  }
}
```

**返回参数说明**

| 参数名 | 类型 | 说明   |
| :----- | :--- | ------ |
| code   | int  | 状态码 |

**备注**

- 更多返回错误代码请看首页的错误代码描述





# 巡检任务信息查询接口

**简要描述：**

- 查询巡检检查任务和异常任务

**请求URL：**

- `URL/api/get_task_numberr`

**请求方式：**

- POST

| 参数名      | 必选 | 类型   | 说明                                  |
| ----------- | ---- | ------ | ------------------------------------- |
| Token       | 是   | String | 密钥                                  |
| uid         | 是   | Int    | 用户id                                |
| page_number | 是   | Int    | 页码（页码从0开始，每页限制返回10条） |
| category    | 是   | String | 任务类型（checkTask，checkUnusual）   |

```json
[
    {
        "employee_id": [
            7,
            "陈昊"
        ],
        "check_type": "集团公司专项检查",
        "show_is_special_str": "专项检查",
        "name": "Task2020060003",
        "check_notes": "2020-07-10 整改",
        "company_name": "上海宝房（集团）有限公司",
        "safety_point_id": [
            6,
            "上海市宝山区机关事务管理局"
        ],
        "id": 3,
        "product_id": [
            363,
            "泰和路245号3室"
        ]
    }
  ]
```

| 返回参数            | 类型   | 说明         |
| ------------------- | ------ | ------------ |
| id                  | int    | 任务id       |
| product_id          | int    | 租赁单元     |
| safety_point_id     | int    | 巡检点       |
| company_name        | string | 公司名       |
| check_notes         | string | 检查结果     |
| name                | string | 任务编号     |
| show_is_special_str | string | 是否专项任务 |
| check_type          | string | 检查类型     |
| employee_id         | int    | 巡检员       |
